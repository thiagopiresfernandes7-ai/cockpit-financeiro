Como usar em 5 minutos

1
Comece pelo Painel

Ele mostra sua situação geral: saldo, gastos, patrimônio e a próxima ação recomendada.

2
Use Registrar para tudo

Lance compras, receitas, investimentos, dívidas e saldos pela mesma tela. Menos páginas, menos esforço.

3
Planeje só o essencial

Cadastre limites de gasto e metas. O app mostra alertas quando você se aproxima do limite.

4
Veja Carteira quando quiser detalhe

Investimentos, dívidas e simulações ficam juntos para você não se perder.

5
Faça backup

Use Exportar para salvar seus dados. Use Importar para restaurar depois.

Privacidade

Os dados ficam salvos apenas neste navegador. O app não pede senha bancária e não acessa banco automaticamente.

Recomeçar

Se quiser começar do zero, apague os dados locais. Os dados exemplo podem voltar pelo botão no topo.
